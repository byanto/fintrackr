# 💸 fintrackr

A full-stack personal finance management app built with **Spring Boot** and **React**, designed to help users track small business inventories, transactions, and investment portfolios — all in one place.

---

## 🧠 Project Goals

I created **fintrackr** as a portfolio project to:

- Reinforce my backend development skills using **Java 21** and **Spring Boot 3**
- Learn and implement **RESTful API design**
- Practice real-world Git workflows with GitHub
- Apply modern backend architecture with DTOs, validation, and persistence
- Demonstrate my ability to build scalable backend systems from scratch

---

## ⚙️ Tech Stack

| Layer         | Technology               |
| ------------- | ------------------------ |
| Backend       | Java 21, Spring Boot 3   |
| Database      | H2 (Dev) / PostgreSQL    |
| Build Tool    | Maven                    |
| Versioning    | Git & GitHub             |
| IDE           | VSCode                  |
